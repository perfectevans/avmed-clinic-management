---

# 📌 Overview
The **AVMED Clinic Management System** solves the problem of cluttered and complex medical software. By using a minimalist "Zen" interface, it reduces cognitive load for medical staff while ensuring 100% data integrity for patient records.

# 🎯 Key Features
* **Patient Onboarding:** Securely register and track patient demographics.
* **Appointment Scheduling:** Conflict-free booking system.
* **Medical History Logs:** Persistent storage of diagnoses and treatments.
* **Zen Dark UI:** A high-contrast, low-strain interface designed for long shifts.

# ⚙️ Technical Stack
* **Frontend:** Python (CustomTkinter/Tkinter) for a modern, responsive Dark Mode UI.
* **Backend:** SQL (SQLite) for lightweight, ACID-compliant data storage.
* **Security:** Role-based access logic for patient privacy.

# 🚀 Installation & Setup

Follow these steps to set up the **AVMED Clinic Management System** on your local machine:

1. **Clone the repository:**
   ```bash
   git clone [https://github.com/perfectevans/avmed-clinic-management.git](https://github.com/perfectevans/avmed-clinic-management.git)

2. **Install requirements:**
Ensure you have Python 3.11+ installed, then run:

   ```bash
   pip install -r requirements.txt
   ```

3. **Launch the system:**
   ```bash
   python app.py
